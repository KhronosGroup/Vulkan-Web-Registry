Name

vkGetPhysicalDeviceXcbPresentationSupportKHR — query physical device for presentation to X11 server using XCB.

C Specification

Parameters

physicalDevice
A physical device handle.
queueFamilyIndex
Index to a queue family.
connection
Pointer to a xcb_connection_t.
visual_id
An xcb_visualid_t.

Description

vkGetPhysicalDeviceXcbPresentationSupportKHR determines whether a queue family of a physical device supports presentation to an X11 server, using the XCB client-side library.
