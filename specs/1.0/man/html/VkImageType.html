C Specification

The basic dimensionality of an image is specified by VkImageCreateInfo::imageType, which must be one of the values

typedef enum VkImageType {
    VK_IMAGE_TYPE_1D = 0,
    VK_IMAGE_TYPE_2D = 1,
    VK_IMAGE_TYPE_3D = 2,
} VkImageType;

Description

These values specify one-, two-, or three-dimensional images, respectively.

VkImageCreateInfo, vkGetPhysicalDeviceImageFormatProperties, vkGetPhysicalDeviceSparseImageFormatProperties
