Name

VkCommandBufferAllocateInfo — Structure specifying the allocation parameters for command buffer object.

C Specification

typedef struct VkCommandBufferAllocateInfo {
    VkStructureType         sType;
    const void*             pNext;
    VkCommandPool           commandPool;
    VkCommandBufferLevel    level;
    uint32_t                commandBufferCount;
} VkCommandBufferAllocateInfo;

Fields

sType
Structure type. Must be VK_STRUCTURE_TYPE_COMMAND_BUFFER_ALLOCATE_INFO.
pNext
Pointer to next structure in the structure chain when applicable.
commandPool
The pool from which to allocate the command buffers.
level
The level of the command buffers to allocate.
commandBufferCount
The numbef of command buffers to allocate.

Description

This structure is used to specify the parameters of command buffer objects allocated using vkAllocateCommandBuffers.

Valid Usage

  • sType must be VK_STRUCTURE_TYPE_COMMAND_BUFFER_ALLOCATE_INFO
  • pNext must be NULL
  • commandPool must be a valid VkCommandPool handle
  • level must be a valid VkCommandBufferLevel value
