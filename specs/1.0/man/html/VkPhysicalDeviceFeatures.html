Description

The VkPhysicalDeviceFeatures structure contains a feature flag for each of the fine-grained features that may be supported by an implementation.

When passed to vkGetPhysicalDeviceFeatures as the pFeatures parameter, the implementation will fill in each member of the structure with VK_TRUE if the indicated physicalDevice supports the feature, or with VK_FALSE if the physical device does not support the feature.

Fine-grained features must be enabled at VkDevice creation time. This is done by passing a pointer to a VkPhysicalDeviceFeatures structure in the pEnabledFeatures member of the VkDeviceCreateInfo structure that is passed to vkCreateDevice. In this case, setting a member of the structure to VK_TRUE will enable support for the feature on the indictated physical device, and setting a member to VK_FALSE will disable support for the feature.
